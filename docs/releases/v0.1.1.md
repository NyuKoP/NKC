# NKC v0.1.1

## Summary

- Fixed right panel tab layout so inactive tab content no longer occupies space.
- Improved friend add and route-ready UI feedback for onion-based routing.
- Hardened media download handling and basic media payload validation.
- Prevented trust state from auto-promoting to verified on connect alone.

## UI

- Aligned `About`, `Media`, and `Settings` tab container behavior in the right panel.
- Fixed the `Settings` tab being pushed downward by inactive `Media` content.
- Improved friend code/loading hints and route-ready retry behavior.

## Security and Stability

- Sanitized downloaded media filenames.
- Delayed object URL revocation to avoid premature download failures.
- Rejected suspicious image/video payloads when MIME and content do not match expected signatures.
- Kept trust state conservative unless verification state is explicitly established.
