# NKC v0.3.0 - Go Transport Core

## Highlights

- Moved SOCKS5 negotiation and HTTP/HTTPS forwarding from Electron TypeScript into the Go worker.
- Added shared connection pooling, bounded concurrency, response limits, retries, and proxy authentication in Go.
- Moved route validation, Lokinet-to-Tor failover, and failed Tor delivery queueing behind `transport.forward`.
- Kept the authenticated loopback controller, local inbox, IPC, and diagnostic publication in Electron.
- Removed the duplicate TypeScript socket, HTTP parser, TLS, retry, and agent-pool implementations.
- Disabled automatic redirects in native overlay requests to prevent unintended network target changes.

## Validation

- TypeScript node and renderer typechecks
- Full ESLint and UTF-8 validation

- Production Vite/Electron build
